MyPass - Password ManagerMyPass is a Python-based desktop application that allows users to securely store, retrieve, and generate passwords for various websites. It provides a user-friendly graphical interface built with Tkinter.Core FeaturesAdmin Login Authentication: Secures the application behind an administrative password, preventing unauthorized access.Cryptographic Storage: Safely stores passwords in a JSON file using Fernet symmetric encryption to ensure credentials are not exposed in plaintext.Automated Password Generation: Generates highly secure, randomized passwords utilizing a mix of uppercase letters, lowercase letters, numbers, and special symbols.Credential Retrieval: Allows users to efficiently search and retrieve stored passwords by querying the website name.Comprehensive Audit View: Provides an interface to view a complete list of all saved password entries.Visibility Controls: Includes a checkbox to toggle password visibility (show/hide) during data entry.System Architecture & TechnologiesPython: Core programming language utilized for application logic.Tkinter: Standard Python GUI library used for rendering windows, buttons, and input fields.JSON: Used as the lightweight database schema for structured data storage.Cryptography (Fernet): Implements AES encryption to protect stored credentials.Pyperclip: Utilized for automated clipboard operations.Installation & Setup Instructions1.
